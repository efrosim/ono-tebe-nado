# Оно тебе надо

Одностраничный макет аукциона «Оно тебе надо»: шапка на grid, карточки лотов на flex с переносом, секция «Об аукционе» на grid, общий overlay и переиспользуемый блок address.

## О проекте

- Семантическая разметка: header, main, footer, section, nav, article, address.
- Единый заголовок h1, корректные списки ul/li, ссылки tel:/mailto:.
- Порядок подключений: fonts → styles/global.css → styles/style.css.

## Технологии

- HTML5, CSS3 (Grid/Flex), БЭМ‑наименование классов (без строгого БЭМ‑дерева).
- Переиспользуемые компоненты: overlay, address, карточка card + модификаторы фона.

## Структура

- index.html
- styles/global.css
- styles/style.css
- fonts/ (fonts.css, файлы шрифтов)
- images/ (изображения, иконки соцсетей, логотипы)

## Запуск

Открой index.html в браузере или подними статический сервер IDE/расширением.

## Соответствие чек‑листу

- Семантика, валидность HTML, порядок подключений CSS, отсутствие горизонтального скролла.
- Grid в header/footer, Flex для списков/карточек, перенос карточек при переполнении.
- Переиспользование стилей overlay/address/карточек и отсутствие инлайновых стилей.
